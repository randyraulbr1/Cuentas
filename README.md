# Cuentas Claras

App web (PWA) para controlar tu ingreso mensual, pagos fijos, tarjetas de crédito y ahorro.

- Sin backend: todo se guarda en el propio dispositivo (localStorage).
- Instalable como app desde el navegador ("Añadir a pantalla de inicio").
- Sugiere qué tarjeta pagar primero (método avalancha, mayor interés primero) y cuánto ahorrar cada mes.
- Guarda un historial mensual con el estado de cada mes (correcto / apretado / peligroso).

## Uso

Abre `index.html` publicado en GitHub Pages, o instálalo como app en tu teléfono desde ahí.
